# Spotify Wrapper

A wrapper to work the [Spotify Web API](https://developer.spotify.com/documentation/web-api/)

### Installing

```
git clone https://github.com/murilosandiego/js-tdd-boilerplate.git
cd js-tdd-course
rm -rf .git
git init
npm run test:coverage
```
